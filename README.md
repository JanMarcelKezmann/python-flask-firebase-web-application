# Flask Firebase Web application with Python

## Getting started

- Clone this repo 
- Install requirements
- Run the script
- Check http://localhost:5000
- Done! :tada:


## Local Installation

### Clone the repo
```shell
git clone https://github.com/JanMarcelKezmann/python-flask-firebase-web-application.git
```

### Install requirements

```shell
pip install -r requirements.txt
```

Make sure you have the following installed:
- tensorflow
- keras
- flask
- pillow
- h5py
- gevent
- flask_dropzone
- pyrebase
- numpy

### Run with Python (3.6)

Python 3.6 is supported and tested.

#### For Linux
```shell
python app.py
```

#### For Windows
```cmd
python app.py
```

### Try it out!

Open http://localhost:5000 or http://127.0.0.1:5000 and have fun. :smiley:
